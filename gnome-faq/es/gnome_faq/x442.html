Ya tengo este módulo, ¿cómo me deshago de todos los ficheros object y otras "cruft" del árbol del fuente?

Tienes tres opciones, make clean , make distclean o borrar el directorio. Usar make clean borrará todos los object files, built executables, y miscellaneous cruft, es sencillo y seguro. La siguiente opción, make distclean debería borrar todo lo que no estubiera en el tarball original. Si haces esto, no podrás desinstalar el módulo . Y lo mismo para borrar el source directory. Basicamente, make clean es bueno para un uso general, las otros deberían ser consideradas medidas drásticas.
