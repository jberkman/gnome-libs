De acuerdo, ahora tengo un Makefile. ¿Cómo consigo instalar GNOME en mi sistema?

Desde aquí, todo es bastante sencillo. Escribes make , seguido de make install . En muchos sistemas (incluido GNU/Linux), esto debería ser seguido de ldconfig . Si tienes mucha prisa, a veces puedes saltar make , ya que make install normalmente hará todo lo que no halla sido hecho todavía. Si quieres empezar todo el proceso y olvidarte de ello, y usas la Bourne shell o sus derivadas (p.ej. ksh o bash), pon los cuatro comandos en una línea separados por && . Por ejemplo: 

$
./configure --prefix=/usr/local && make && make install && ldconfig

El && se asegura de que el primer comando ha finalizado correctamente antes de seguir con el siguiente.

Una vez que has hecho make install (y ldconfig si fuera necesario), puedes continuar con el siguiente paquete.
