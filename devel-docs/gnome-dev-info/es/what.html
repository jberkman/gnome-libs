Que husmear.

Si te quieres manchar las manos con GNOME, tengo algún consejo para ti:

  • Comienza con cosas simples. Haz pequeñas cosas y familiarízate con GNOME. Ya tendrás tiempo de hacer grandes cosas después.

  • Lee el TODO. Encontrarás algunas cosas interesantes sobre las que trabajar.
