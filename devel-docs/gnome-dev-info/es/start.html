Como empezar a trastear con GNOME.

Lee los ficheros README y HACKING para saber como bajarte la última versión de GNOME desde el CVS.

Necesitarás conocer la herramienta GTK+. Si no estas familiarizado con ella debes considerar leer el Gtk+ Tutorial.

Cuando has hecho tus cambios nesitas hacer un diff... La mejor manera de hacer esto es:

        $ cvs diff -u > diffs
        

[FIXME:¿Funciona con CVS anónimo?] Después de esto envía el fichero diffs a <gnome-list@gnome.org>. En un corto espacio de tiempo uno de los hackers del núcleo te responderá y él va a ser tu "padrino GNOME". Entonces, y hasta que tengas una cuenta CVS, manda tus parches a él.
