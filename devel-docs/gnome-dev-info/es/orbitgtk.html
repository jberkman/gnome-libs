ORBit-Gtk

Autor(es)

Elliot Lee <sopwith@redhat.com>

Descripción

Rutinas para hacer que ORBit trabaje elegantemente con gtk, y arranque/recupere el servicio de nombres de objetos para una sesión de GNOME.

gnome_CORBA_init inicializa ORBit y GNOME para ser usados en un programa.

CORBA_ORB gnome_CORBA_init(char *app_id, struct argp *app_parser, int *argc, char **argv, unsigned int flags , int *arg_index, CORBA_Environment *ev);

Descripción

Esta rutina inicializa gnome (llamando a gnome_init). Unos pocos items serán establecidos para permitir que ORBit y gtk interactúen bien, y finalmente inicializa ORBit y devuelve la referencia CORBA_ORB al nuevo ORB.

Uso

int main(int argc, char *argv)
{
	CORBA_ORB my_orb;
	CORBA_Environment ev;

	CORBA_exception_init(&ev);
	my_orb = gnome_CORBA_init("myapp", NULL, &argc, &argv, 0, NULL, &ev);
	if(ev._major != CORBA_NO_EXCEPTION)
	    exit(1); /* error initializing ORB */

	/* inserta el código para matar a la aplicación aquí */
}

Parámetros

  • char *app_id

    Una cadena de caracteres alfanúmerica de ID (identificación) de la aplicación (el nombre de la aplicación es una buena elección).

  • struct argp *app_parser

    Ver los (inexistentes) documentos de argp como ayuda .

  • int *argc

    Un puntero al número de argumentos.

  • char **argv

    El vector de cadenas de argumentos.

  • unsigned int flags

    Ver los (inexistentes) documentos de argp como ayuda.

  • int *arg_index

    Ver los (inexistentes) documentos de argp como ayuda.

  • CORBA_Environment *ev

    Puntero a una estructura de información de excepción de CORBA, para permitir a las funciones de inicialización del ORB señalar cualquier error que acontezca.

gnome_name_service_get - Recupera una referencia de un objeto al contexto del servicio de nombres raiz para la sesión GNOME actual

CORBA_Object gnome_name_service_get(void);

Descripción

Esta rutina se usa para tener acceso al contexto de nombres más alto para la sesión de GNOME actual. Si ningún servidor de nombres se está ejecutando, se lanzará uno.

Esta función asume que tanto gtk como ORBit se han inicializado.

Uso

CORBA_Object ns;

ns = gnome_name_service_get();
