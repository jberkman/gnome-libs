Funciones utiles para mensajes abstractos

Gnome provee funciones para retransmitir mensajes simples al usuario y preguntar cuestiones simples. Dichas funciones sirven 2 propositos: primero, evitar a los programadore escribir docenas de díalogos idénticos; segundo, permitir al usuario elegir si los mensajes deben aparecer en diálogos o en la barra de aplicación. El conjunto principal de funciones asumen que tu aplicación usa el widget GnomeApp, y que es visible cuando se llaman a estas funciones. Estas funciones tienen un dialog backend y un appbar backend. El dialog backend es expuesto, asi que puedes usarla directamente si no tienes disponible el widget GnomeApp. Las funciones GnomeApp no asumen que tengas un barra de aplicación disponible; deberías usarlas incluso si tu aplicación no tiene appbar. En algunos casos hace una diferencia (por ejemplo, un díalogo puede ser posicionado relativo al GnomeApp).
