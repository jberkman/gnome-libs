Agradecimientos y demás...

Quiero desde aquí animar a cualquier persona que lea este documento, a colaborar con el proyecto Gnome, ya sea desde revisar la traducción de este mismo documento, ortografía, etc. hasta contribuir al repositorio Gnome con código de cálidad.

Este documento se escribió hace algún tiempo, con lo que el núcleo de Gnome puede haber añadido o modificado cosas que aquí no se contemplan.

El siguiente paso de este documento es incluir la totalidad de las librerias (hasta ahora v.1.0), comentandolas y especificando su funcionalidad.

Por último agradecer especialmente la colaboración de Ramón Pons por aguantarme y ayudarme en la traducción de diversas partes de este documento.

Si encuentras algún error en el documento, por favor notificanoslo, si por alguna razón, nuestras cuentas de correo no existiesen, ponerse en contacto con Manuel Vega - barreiro@arrakis.es, el cúal nos lo notificará.

Este documento se encuentra disponible en http://www.croftj.net/~barreiro/spain/gnome
