Funciones proporcionadas por GnomeAppHelper

Lo siguiente es una lista de funciones proporcionadas por GnomeAppHelper.

gnome_app_find_menu_pos - devuelve la posición del elemento del menú. #gnome_app_find_menu_pos - return position of a menu item

GtkWidget *gnome_app_find_menu_pos(GtkWidget *root, gchar *path, gint *pos);

Descripción

Para un elemento del menú especificado por path, devuelve GtkMenuShell del elemento y pone en *pos la posición del elemento dentro de eso, si el elemento se encuentra en el arbol del menú comenzando por la raiz GtkMenuShell y NULL en otro caso. Nuevos menús pueden posteriormente ser insertados después del menu-item con una llamada a: 

          gtk_menu_shell_insert(GTK_MENU_SHELL(parent), new_item, pos);

Uso

	  GnomeApp *app;
          GtkWidget *shell;
          gint pos;

          shell = gnome_app_find_menu_pos(app->menubar, "Edit/Sort/Ascending", &pos);

Parámetros

  • GtkWidget *root

    El shell del menu en la raíz del subarbol menú que debe ser buscado.

  • gchar *path

    Ruta del menú describiendo el elemento a ser buscado.

    • el argumento path debe ser de la forma "Fichero/.../.../NombreItem".

    • "" devolverá la posición que está antes del primer elemento en root.

    • "Fichero/" devolverá la posición de antes del primer elemento del submenú Fichero.

    • "Fichero/Propiedades" devolverá la posición posterior del elemento Propiedades en el submenú Fichero.

    • el uso de "File/<Separator>" debe ser obvio. Sin embargo éste se detiene después del primer separador.

  • gint *pos

    El apuntador a entero puesto por este parámetro es la posición del elemento menú descrito por path en el shell del menu del padre.

gnome_app_remove_menus - borra un número de elementos del menú.

void gnome_app_remove_menus(GnomeApp *app, gchar *path, gint item_count);

Descripción

Esta función borra item_count items de la estructura de menu de GnomeApp, comenzando con el item descrito por path.

Uso

            gnome_app_remove_menus(app, "Edit/Sort/Ascending", 2);

Parámetros

  • GnomeApp *app

    El widget GnomeApp con la barra de menú que contiene los elementos a ser eliminados.

  • gchar *path

    Ruta describiendo el primer elemento a ser borrado. Ver gnome_app_find_menu_pos para más información sobre rutas de menús.

  • gint item_count

    Número de elementos consecutivos a ser eliminados.

gnome_app_insert_menus - inserta una estructura de menú en una barra de menú de GnomeApp

void gnome_app_insert_menus(GnomeApp *app, gchar *path, GnomeUIInfo *uiinfo);

Descripción

Inserta una estructura de menú descrita por uiinfo en la posición descrita en path.

Uso

	    static GnomeUIInfo edit_menu[] = {
	    { GNOME_APP_UI_ITEM, N_("_Copy"), N_("Copy text"), copy_callback, NULL, NULL,
	      GNOME_APP_PIXMAP_STOCK, GNOME_STOCK_MENU_COPY, 'c', GDK_CONTROL_MASK, NULL },
	    { GNOME_APP_UI_ITEM, N_("C_ut..."), N_("Cut text"), cut_callback, NULL, NULL,
	      GNOME_APP_PIXMAP_STOCK, GNOME_STOCK_MENU_CUT, 'x', GDK_CONTROL_MASK, NULL },
	    { GNOME_APP_UI_ITEM, N_("_Paste"), N_("Paste clipboard contents"), paste_callback, NULL, NULL,
	      GNOME_APP_PIXMAP_STOCK, GNOME_STOCK_MENU_PASTE, 'v', GDK_CONTROL_MASK, NULL },
	    GNOMEUIINFO_END
	    };

	    static GnomeUIInfo more_menus[] = {
	    { GNOME_APP_UI_ITEM, N_("_Search..."), N_("Search for text"), search_callback, NULL, NULL,
	      GNOME_APP_PIXMAP_NONE, NULL, 's', GDK_CONTROL_MASK, NULL },
	    { GNOME_APP_UI_ITEM, N_("_Replace..."), N_("Replace text"), replace_callback, NULL, NULL,
	      GNOME_APP_PIXMAP_NONE, NULL, 'r', GDK_CONTROL_MASK, NULL },
	    GNOMEUIINFO_END
	    };

	    static GnomeUIInfo main_menu[] = {
	    GNOMEUIINFO_SUBTREE (N_("_Edit"), edit_menu),
	    GNOMEUIINFO_END
	    };

            app = gnome_app_new("app", "App");
            gnome_app_create_menus(app, main_menu);

	    /* Hacer algo */ 

	    /* insertar elementos Search y Replace después de Paste */
            gnome_app_insert_menus(app, "Edit/Paste", sort_menu);

Parámetros

  • GnomeApp *app

    GnomeApp contiene la barra de menú en la que los nuevos elementos serán insertados.

  • gchar *path

    Ruta describiendo la posición donde los nuevos elementos serán insertados. Ver gnome_app_find_menu_pos para más información sobre rutas de menú.

  • GnomeUIInfo *uiinfo

    Un puntero al array de entradas GnomeUIInfo describiendo los elementos del menú a insertar.
