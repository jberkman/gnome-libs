gnome-scores - Widget de "Puntuaciones máximas"

Autor(es)

Horacio J. Peña <horape@compendium.com.ar>

Éste es software libre (bajo terminos de la licencia GNU LGPL)

Descripción

Es un widget especializado en mostrar las "Puntuaciónes Máximas" en juegos. Está muy integrado con el componente gnome-score así que solo necesitas llamar una función para hacer todo el trabajo...

Uso

  Usa la función gnome_scores_display. Las otras funciónes van a ser 
  interrumpidas...(vale, antes debería añadir soporte para mapas de pixels a 
  * _display)

gnome_scores_display - Realiza todo el trabajo de mostrar los mejores marcadores.

void gnome_scores_display(gchar *title, gchar *app_name, gchar *level, int pos);

Descripción

Llama a la función gnome_score_get_notables para recuperar la información, crear la ventana y mostarla.

Retorno

No devuelve nada

Uso

pos = gnome_score_log (score, level, TRUE);
gnome_scores_display (_ ("Mi game"), "migame", level, pos);

Parámetros

  • gchar *title

    Titulo.

  • gchar *app_name

    Nombre de la aplicación, como en gnome_score_init.

  • gchar *level

    Nivel del juego o NULL.

  • int pos

    Posición entre los diez mejores en la que se encuentra el jugador actual, devuelta por gnome_score_log.

gnome_scores_new - Crea la ventana de máximas puntuaciones.

GtkWidget *gnome_scores_new(guint n_scores, gchar **names, gfloat *scores, time_t *times, guint clear);

Descripción

Crea la ventana de máximas puntuaciones. Creates the high-scores window.

Parámetros

  • guint n_scores

    Número de posiciones.

  • gchar **names

    Nombres de los jugadores.

  • gfloat *scores

    Puntuaciones

  • time_t *times

    Hora en la que se realizó la puntuación

  • guint clear

    ¿Añade un botón de "Limpiar"?
