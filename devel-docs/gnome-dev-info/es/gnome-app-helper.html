Introducción

El módulo GnomeAppHelper permite crear facilmente menús y barras de herramientas para la aplicación. En vez de tener que crear todos los widgets requeridos a mano, GnomeAppHelper te permite crear simplemente un vector de definiciones para los items que tu quieres en tu menú o barra de herramientas. Puedes llamar a las funciones que cambian este array en todos los widgets necesarios automáticamente. El widgets tendrá todos los aceleradores de teclado y teclas rápidas configurados correctamente para tí.

Además para la creación normal de items de menús y barra de herramientas, GnomeAppHelper permite crear rapidamente items auxiliares que trabajen como botones de chequeo, items de opción, y listas de entradas al menú de ayuda

Debes usar GnomeAppHelper cuando quieras crear en tu aplicación barras de menú, barras de herramientas o menús despegables. GnomeAppHelper permite usar un mecanísmo de iconos almacenados para dar a tus widgets un aspecto consistente.
