Qué es GNOME..

GNOME significa 'GNU Network Object Model Environment', entorno de trabajo en red orientado a objetos. El proyecto GNOME tiene la intención de construir un completo y amigable escritorio basado enteramente en software libre. GNOME forma parte del proyecto GNU. El escritorio consistirá en pequeñas utilidades y grandes aplicaciones las cuáles compartirán un aspecto y una percepción consistente. GNOME usa GTK como herramienta GUI para todas las aplicaciones que cumplen las premisas dictadas en GNOME (gnome-compliant).
