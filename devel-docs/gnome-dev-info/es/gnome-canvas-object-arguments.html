Atribuos y argumentos de objetos

El canvas hace uso extensivo del sistema de argumentos de objetos de GTK — todos los atributos de los items son configurados y consultados a través de él. Solo necesitas tener un minimo conocimiento de como el sistema de argumentos funciona, por lo tanto no te preocupes si no te quieres aprender todos los detalles. Aquí explicaremos todo lo que necesitas saber.

Usar el sistema de argumentos de Gtk para configurar y consultar sobre los argumentos de los items del canvas tiene la ventaja de facilitar su trabajo a las envolturas con los lenguajes, y también permitir tener muchos puntos de entrada al API para todos lo items del canvas.
