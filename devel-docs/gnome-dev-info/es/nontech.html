Lenguajes de Programación que pueden ser usados en GNOME.

GNOME tiene la intención de soportar cualquier lenguaje. Pero necesitará las envolturas GTK+ y GNOME para su lenguaje particular. Nuestros lenguajes preferidos son C y Scheme (usando guile). También hay envolturas para Objective C. Hay ligaduras GTK para C++ (Gtk--) y para Perl, pero por lo que sé, no son muy buenas todavía y no hay ligaduras para el material de GNOME para estos lenguajes.
