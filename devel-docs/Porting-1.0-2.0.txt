		Porting GNOME 1.0 Applications to GNOME 2.0

Using new feature: libgnome/gnomelib-init2.h

- The new program initialization system allows the application to
  specify the modules it will use, and parameters to be passed to those
  modules to affect their initialization.  It replaces the old
  gnome-init style.

New functions:
- gnome_message_box_get_label

Removed functions:
- gnome_message_box_set_modal
  (use gnome_dialog_set_modal instead)
- gnome_message_box_set_default
  (use gnome_dialog_set_default instead)

Removed Modules:
- gnome-font-selector
- gnome-guru (use gnome-druid instead)
- gnome-spell
- gnome-startup (use gnome-session instead)
- gtkcauldron (use glade instead)
